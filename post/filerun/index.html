🌈FileRun网盘安装

Self-Hosted File Sync and Sharing

环境概述

  • CentOS 7
  • 宝塔面板
  • php5.6
  • MySQL

安装步骤

下载安装包

定位到FileRun官网-Download，下载最新的安装包。

上传至服务器

在宝塔面板里上传压缩包至网站根目录，解压缩

排错

注意需要安装诸多PHP的插件

同时也要启用PHP configuration: "exec"，在宝塔面板的软件中找到PHP，点开安装扩展，将图上需要用到的扩展全部安装。

定位到左侧配置修改，搜索enable_dl字段，将off改为on，搜索disabled_functions，删除里面的exec字段并重启php服务即可。

开始安装

输入你的域名，FileRun会自动进入安装界面

检查你的组件和配置是否都安装完毕。

对于open_basedir的设置打开宝塔面板左侧的配置文件，找到open_basedir字段，修改为open_basedir = /www/wwwroot/yourdomain.com/:/tmp/:/proc/

点击下一步开始配置MySQL数据库。

输入对应的用户名，数据库名和密码，下一步即可。

记住你的账户和密码，之后即可登录
