🌈Update AzureDevops Server Certificates

🌈Update AzureDevops Server Certificates

2019-05-09 1 min read

简单记记

获取pfx证书

如果你的证书已经是pfx格式了，那么可以直接复制到服务器待用

如果你的证书是.cer格式那么还需要将格式转化为pfx

可以在https://freessl.cn/ssltools找到SSL证书格式转换

更换证书

登录到你的服务器

win+r输入certlm.msc打开证书管理控制台

删掉原先导入的服务器证书以免冲突，便于识别

打开IIS控制台

依次点开左侧网站，找到我们要更换证书的Azure Devops Server

右侧找到编辑网站-绑定

编辑443及HTTPS端口的设置

选择我们正确的证书和主机地址即可

更换完成之后，打开我们的主机名

可以看到SSL证书已经更换成功
