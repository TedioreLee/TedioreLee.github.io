🌈Update Domain Password Policy

· 2019-04-15 · # Windows

前言：

公司的账号总是有问题，有的人个把月就要求修改密码，有的人很久都不用修改密码，故本次重新定义密码策略，规范一下统一的密码策略。

步骤

在DC上打开组策略管理

双击Default Domain Policy，在右侧能看到你组织的当前所有策略。

在Default Domain Policy上右键，编辑，可以编辑当前组策略

依次展开，策略-Windows设置-安全设置-账户策略-密码策略

即可修改你想要的组密码策略，这里以我个人举例
