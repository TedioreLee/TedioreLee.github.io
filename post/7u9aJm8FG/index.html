Dokcer是个好东西

  • Docker pull images:$docker pull images:tag(default tag name is 'latest')
  • Docker list all images:$docker images
  • Docker start a image:$docker run -itd ubuntu
  • Docker list all running image:$docker ps -a
  • Docker attach a running image:$docker attach id
  • Docker dettach from a running image:ctrl+p+q
